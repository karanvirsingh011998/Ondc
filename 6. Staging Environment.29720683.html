6. Staging Environment

Staging, as the name defines is the first integration point to get you on-boarded to the network.

ONDC Team has created a sandbox environment where all early stage network participants can test end-to-end transactions.

For staging environment, the network participants need to complete the end-to-end testing with ONDC reference applications.

Being a seller Network participant, kindly test the end-to-end transaction flow with ONDC Reference Buyer App.
